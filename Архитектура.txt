    Этапы работы приложения

1. Приветствие

2. Чтение из .ini файла

    а) (если первый раз) вывод справки


3. Суперцикл для работы приложения: обработка команд

    а) предложение начать новую партию (или ввести другую команду)

    б) (если нет текущего игрока) запрос имени игрока

    в) предложение выбрать режим игры

        если одиночная:

            i) (если есть save) предложить загрузить сохранённую партию

            ii) (если нет save) выбор сложности

            iii) (если нет save) выбор символа

            iv) старт партии
                вложенный цикл для обработки ввода ходов для партии

        если для двоих:

            i) запрос имени второго игрока

            ii) (если есть save) предложить загрузить сохранённую партию

            iii) (если нет save) выбор символа для первого игрока

            iv) старт партии
                вложенный цикл для обработки ввода ходов для партии

    г) вывод статистики для текущего(-их) игрока(-ов)
